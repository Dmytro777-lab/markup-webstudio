WebStudio

A simple landing page for a digital marketing and web design studio.

Description

This project is a static website created with HTML and CSS. The page includes
the following sections:

- Header with navigation and contacts
- Hero section
- Goals / Features section
- Our Team section
- Portfolio section
- Footer

Technologies Used

- HTML5
- CSS3
- Flexbox
- Modern Normalize
- Google Fonts
