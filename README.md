# Markup WebStudio

A responsive landing page for a digital marketing and web design studio.

## Description

This project is a static website built with HTML and CSS.

### Features

- Header with navigation and contacts
- Hero section
- Features section
- Our Team section
- Portfolio section
- Footer

## Technologies

- HTML5
- CSS3
- Flexbox
- Modern Normalize
- Google Fonts
